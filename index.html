About ACOE
The Alameda County Office of Education, under Superintendent Alysse Castro, operates schools serving the county's most vulnerable students — including Court Schools at the JJC, Quest Academies, Community Schools, and the Parenting Teens Program.
www.acoe.org ↗
About AC Probation
The Alameda County Probation Department, under Chief Brian K. Ford, provides supervision, intervention, and support for justice-involved youth through its Positive Youth Development division — including YVRC, the Ubuntu Justice Initiative, and the My FUTURE Center.
probation.acgov.org ↗
Our Vision
"We equip the most vulnerable students and those that serve them with the tools to thrive, erasing the predetermination of failure for children from communities of violence and poverty."
Sources: AC Probation (Oakland Youth & County JJCPA), ACOE Schools, City of Oakland DVP

GitHub JSON Connection

This admin panel saves changes to GitHub through the backend function at /api/updateResources. Add your GitHub token as a Vercel environment variable. Do not place the token in this HTML file.

Required Vercel Environment Variables ↓
GITHUB_TOKEN=github_pat_your_token_here GITHUB_OWNER=JohnStyleZ GITHUB_REPO=ProbationResourcesPage GITHUB_BRANCH=main

The token needs GitHub repository Contents permission set to Read and Write.
